class waste_detection_system.transformations.AlbumentationWrapper(albumentation: Callable, format: str = 'pascal_voc')[source]

Bases: Repr

A wrapper for the albumentation package. Bounding boxes are expected to be in xyxy format (pascal_voc). Bounding boxes cannot be larger than the spatial image’s dimensions. Use Clip() if your bounding boxes are outside of the image, before using this wrapper.

class waste_detection_system.transformations.Clip[source]

Bases: Repr

If the bounding boxes exceed one dimension, they are clipped to the dim’s maximum. Bounding boxes are expected to be in xyxy format. Example: x_value=224 but x_shape=200 -> x1=199

Given sets of true and predicted bounding-boxes, determine the best possible match.

Parameters

  • bbox_gt (N1x4 and N2x4 np array of bboxes [x1,y1,x2,y2].) – The number of bboxes, N1 and N2, need not be the same.

  • bbox_pred (N1x4 and N2x4 np array of bboxes [x1,y1,x2,y2].) – The number of bboxes, N1 and N2, need not be the same.

Returns
